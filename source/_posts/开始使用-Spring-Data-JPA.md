Spring-Data-Jpa 是一个开箱即用的 ORM 框架，可以让开发人员很方便地与数据库进行交互。使用 JPA，对于简单的查询，我们几乎可以不用写一行 SQL 即可完成 CRUD 功能。
```
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>
```
